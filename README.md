🛠️ PixelForge | Retro Productivity Dashboard

**PixelForge** is a high-performance, pixel-art themed productivity dashboard built for creators who value aesthetics and discipline. It gamifies your daily workflow using an RPG-inspired XP system and a sleek retro-futuristic UI.

## 🚀 Live Link
**Check it out here:** [PixelForge Live Demo](https://pixel-forge-saifou.vercel.app/)

## ✨ Key Features

- **⚔️ Task Forge:** A quest-style task manager with difficulty scaling (Easy, Med, Hard, Boss).
- **🔋 XP & Leveling:** Earn experience points for every completed quest and level up your productivity.
- **⏲️ Focus Core:** A dedicated Pomodoro timer with a retro digital display to keep you in the zone.
- **📊 Stat Board:** Real-time tracking of your completed tasks, total XP, and daily streaks.
- **💾 Data Logs:** Built-in scratchpad for quick notes and technical logs.
- **🌑 Cyberpunk Aesthetics:** A custom-built 8-bit dark mode UI designed for long-night coding sessions.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for optimal accessibility.
- **CSS3:** Custom properties (CSS Variables), Grid, and Flexbox for a responsive pixel-perfect layout.
- **JavaScript (Vanilla):** Core logic for XP calculation, timer mechanics, and DOM manipulation.
- **Local Storage:** All your data persists locally on your machine—no database required.
